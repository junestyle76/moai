# 🗞️ 실시간 뉴스 타임라인

정치, 시사, 경제, 사회 이슈를 타임라인 형태로 제공하는 반응형 웹사이트입니다.

## ✨ 주요 기능

- 📅 년도별/월별 타임라인 구조

- 🏷️ 카테고리별 필터링 (정치/경제/사회/시사)
- 📱 완전 반응형 디자인
- 🔗 외부 링크 새 창 열기
